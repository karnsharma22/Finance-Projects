# Finance Projects Repository

This repository showcases a curated set of financial models and analytical projects built independently to strengthen my applied understanding of valuation, forecasting, and investment analysis. The work here combines structured finance concepts with practical data-driven execution in Excel

---

### **Find:**
- **Valuation Models:** Discounted Cash Flow (DCF) and Comparable Company Analysis for listed Indian firms.  
- **Financial Modelling:** Fully integrated 3-Statement Model (Income Statement, Balance Sheet, and Cash Flow) built from scratch with sensitivity testing.  
- **Analytics & Research:** Mutual Fund performance reports, DuPont and Altman Z-Score analyses, and one-pager company summaries.  
- **Data Tools:** Use of Excel Formulas for financial data cleaning, ratio computation, and visualization.

---

### **Purpose**
These projects are designed to demonstrate proficiency in **financial modelling, equity research, and corporate finance analytics**, while bridging the gap between academic knowledge and real-world applications. Each model reflects how analysts approach valuation and decision-making in a professional finance setting.
